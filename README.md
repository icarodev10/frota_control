# 🚦 FrotaControl

Sistema Fullstack de gestão de frotas desenvolvido como estudo.

https://github.com/user-attachments/assets/17d83fa9-8699-4e2f-9748-fd52b648808c

## 🚀 Tecnologias
- **Back-end:** Python com FastAPI.
- **Front-end:** HTML5, CSS3 (Dark/Vinho) e JavaScript.
- **Banco de Dados:** SQLite.
- **Deploy:** Render (API) e Vercel (Front-end).

## 🛠️ Funcionalidades
- Cadastro de Motoristas e Veículos.
- Registro de Viagens e Infrações.
- Verificação automática de regras de negócio.

## 📁 Estrutura do Projeto
- `/frontend`: Interface do usuário (Vercel).
- `/routes`, `/database`, `/schemas`: Lógica do servidor (Render).
